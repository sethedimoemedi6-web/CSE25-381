What is Blockchain

A blockchain is a digital record(ledger) of transactions

Instead of being stored in one place it is shared across many computers
Data is stored in blocks and these blocks are linked together in a chain hence the term blockchain
Example:think of it like a class notebook where everyone writes transactions one something is written it cannot be erased or changed

History of blockchain

2008:An anonymous person/group going by the pseudoname Satoshi Nakamoto introduced blockchain which was the underlying technology for the crypto currency Bitcoin

2009:Bitcoin was created.
Later:New blockchains like Ethereum added more features like apps. it started with money (Bitcoin) but now it is used for some other things

How does blockchain work

  1. A transaction is requested (e.g sending money)
  2. The transaction is shared with the network
  3. Computers (nodes) verify it
  4. Verified transactions are grouped into a block
  5. The block is added to the chain forming a block of chains called blockchain
  6. Transaction is complete

Blockchain consesus Mechanism:POW && POS

Proof of work

Pow Proof of Work (PoW) is a decentralized consensus mechanism in blockchain that requires participants (miners) to solve complex cryptographic puzzles to validate transactions and create new blocks. It secures networks like Bitcoin against fraud and double-spending by requiring high computational power, preventing, in turn, tampering with data

Proof of stake

What Is Proof of Stake (PoS)? | The Motley FoolProof of Stake (PoS) is a blockchain consensus mechanism that selects validators based on the number of cryptocurrency tokens they hold and "stake" as collateral. Unlike energy-intensive mining, PoS offers a faster, more energy-efficient, and scalable way to confirm transactions and create new blocks.It is used by the world second largest crypto currency ethereum

Blockchain network structure :Nodes and Forks

Nodes

Nodes are computers within a blockchain network that maintain the distributed ledger, validate transactions, and communicate updates, acting as the backbone of decentralization

Forks

Forks are structural, protocol-level changes or divergences that occur when blockchain rules are updated or split, leading to new, often incompatible, versions.
